## Usage
##### Open terminal, locate to the directory

```bash
cd folder_name

```
##### Open File in your IDE or any text editor

```bash
vim *.h
vim *.c
```
###### replace * (asterisk) with file name
###### See and comprehend...

##### Compile code (.c file)
```bash
gcc *.c *.out
```
###### replace * (asterisk) with file name

##### Run compiled code (.out file)
```bash
./*.out
```
